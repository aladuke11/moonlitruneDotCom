Welcome to the CryptoCorp Alternate Reality Game

This is a gamified experience embedded within the portfolio website. Complete the steps below to unlock secrets, reduce debt, and discover hidden content!

Step 1: Visit the Cyberpunk Store

Objective: Make your first purchase

  • Navigate to the Cyberpunk Store
  • Click "Buy Now" on any product (Jacket, Wristband, or Backpack)
  • A popup will appear asking if you want to borrow credits
  • Click "Agree" to complete the purchase
  • Result: You incur a $500 debt with CryptoCorp

Step 2: View Your Debt on the Home Page

Objective: See the debt tracking system

  • Return to the home page
  • A debt bar will appear at the top showing your current CryptoCorp debt
  • The bar displays: debt amount, progress bar, and a Reset button
  • Result: You can now see the financial consequences of your purchases

Step 3: Unlock HoloVision Goggles (Secret)

Objective: Find and activate the secret unlock

  • Go to the LinkTree Clone page
  • Scroll down to find the "Unlock HoloVision Goggles" button
  • Click the button to unlock the goggles in your account
  • Return to the store and notice the HoloVision Goggles product is still sold out
  • Result: Goggles are marked as unlocked in your save data

Step 4: Purchase HoloVision Goggles

Objective: Buy the locked product

  • Go back to the Cyberpunk Store
  • The HoloVision Goggles should now be available (no longer sold out)
  • Click "Buy Now" on the goggles
  • Agree to the credit terms (+$500 debt)
  • Result: You now have the goggles and increased debt

Step 5: Discover the Hidden Box

Objective: Unlock the secret message

  • After purchasing the goggles, a hidden box should appear on the store page
  • Click "Close Access" to dismiss the hidden message
  • The hidden box contains augmented reality protocol information
  • Result: You've accessed restricted systems through the goggles

Step 6: Use the HACK Easter Egg

Objective: Reduce debt using the goggles

  • Return to the home page with active debt and goggles unlocked
  • A green "‚ö° HACK" button will appear on the debt bar
  • Click the HACK button to reduce your debt by 10%
  • You can use this repeatedly to gradually clear your debt
  • Result: The goggles grant you the ability to hack the CryptoCorp system

Step 7: Activate Code Vision

Objective: See the website's code through the goggles

  • With goggles unlocked, a cyan "üëÅÔ∏è CODE VISION" button appears in the bottom right
  • Click the button to toggle Code Vision mode

  • Result: HoloVision reveals the underlying structure of reality (the website code)!

Step 8: Complete Your Debt Freedom (Optional)

Objective: Clear all debt

  • Keep clicking the HACK button to reduce debt by 10% each time
  • Each hack reduces your remaining balance incrementally
  • When debt reaches $0, the debt bar disappears and you're free
  • Alternate: Click the Reset button to immediately clear all progress
  • Result: Victory - you've escaped CryptoCorp's debt system!

üéÆ Game Mechanics Summary

Debt System

  • Each purchase adds $500 to your total debt
  • Debt is displayed in a progress bar at the top of the home page
  • Debt persists across sessions (stored in localStorage)
  • Maximum debt visualized: $2,000 (shown as 100% on progress bar)

Unlock System

  • HoloVision Goggles start as "sold out" on the store
  • Secret unlock button on LinkTree page enables purchase
  • Unlocking is permanent (stored in localStorage)
  • Goggles unlock special easter egg features

Easter Eggs & Secrets

  • HACK Button: Only visible with goggles + active debt
  • Reduction Amount: 10% of current debt per hack
  • Hidden Box: Appears after goggle purchase on store
  • Augmented Reality Message: Immersive lore about restricted systems
  • Code Vision: Toggle button to see website source code overlays
  • Code Snippets: Floating HTML, CSS, and JS code reveals page structure

Reset Function

  • Red "Reset" button on debt bar (always visible when debt exists)
  • Requires confirmation (prevents accidental clicks)
  • Clears: All debt, all purchases, all unlocks
  • Allows you to restart the ARG from the beginning
