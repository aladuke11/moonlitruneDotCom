üß† Quick Quiz Time!

Q1: Which pseudo-class would you use to style every other row in a table?

Answer: :nth-child(even) or :nth-child(odd)
This creates the "zebra stripe" effect commonly seen in tables!

Q2: How do you select the first paragraph in a div?

Answer: div p:first-child
This selects the first paragraph that is a child of a div!

Q3: What pseudo-class makes buttons interactive when users click them?

Answer: :active
This triggers when an element is being activated (like clicking a button)!

Q4: What's the difference between :is() and :where()?

Answer: :is() has normal specificity, while :where() has zero specificity.
Use :where() for resets that won't interfere with other styles!

Q5: How do you style a parent element based on its children?

Answer: Use :has() - the "parent selector"
Example: article:has(img) styles articles that contain images!

Q6: What does :nth-child(-n+3) select?

Answer: The first 3 children only
The "-n+3" formula counts backwards from 3: 3, 2, 1!

Q7: What's the correct order for link pseudo-classes?

Answer: LVHA - :link, :visited, :hover, :active
Remember "Love Hate" or "Lord Vader Hates Anakin"!

Q8: How do you style content that appears when a URL fragment matches?

Answer: Use :target
Perfect for creating tab systems and smooth scroll highlights!

6. :is() - Group Similar Elements

Instead of writing long, repetitive selectors, use :is() to group them:

Link Button
/* Instead of this long selector: */ button, a.button, input[type='submit'], input[type='reset'] { /* styles */ } /* Use this clean :is() selector: */ :is(button, a.button, input[type='submit'], input[type='reset']) { background: darkblue; color: white; padding: 10px 20px; border-radius: 5px; } :is(button, a.button, input[type='submit'], input[type='reset']):hover { background: lightblue; }

7. :where() - Zero Specificity Grouping

Like :is(), but with zero specificity - perfect for resets:

8. :has() - Parent Selector (CSS4)

Style parents based on their children - the "parent selector" we've always wanted!

Article with H2

This article has an h2, so it gets special styling!

Article with H3

This article only has an h3, so different styling.

/* Style articles that contain an h2 */ article:has(h2) { border: 3px solid hotpink; background: rgba(255, 20, 147, 0.1); } /* Style articles that DON'T have h2 */ article:not(:has(h2)) { border: 2px solid #4a5568; background: rgba(74, 85, 104, 0.1); }

9. :not() - Exclude Elements

Apply styles to everything EXCEPT specific elements:

/* Style all buttons */ button { padding: 10px 20px; border: none; border-radius: 5px; cursor: pointer; margin: 5px; } /* Primary button gets special color */ button.primary { background: deepskyblue; color: white; } /* All buttons EXCEPT primary get grey */ button:not(.primary) { background: grey; color: white; }

‚ö° Performance & Best Practices

üöÄ Performance Tips:

  • Avoid complex :nth-child patterns on large lists
  • Use :where() for resets to avoid specificity wars
  • Prefer :is() over long selector lists
  • Test :has() browser support before using

üéØ Best Practices:

  • LVHA Order: :link, :visited, :hover, :active
  • Always include :focus for accessibility
  • Use :not() sparingly - can impact performance
  • Combine pseudo-classes for powerful selections
